# Student Data ETL Pipeline

## Project Overview

This project demonstrates a simple ETL (Extract, Transform, Load) pipeline using Python and Pandas.

### Extract
Read student data from a CSV file.

### Transform
Handle missing values by replacing them with the average marks.

### Load
Store the cleaned data into a new CSV file.
